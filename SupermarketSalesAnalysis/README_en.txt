# Supermarket Sales Analysis

This project analyzes a supermarket sales dataset using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Structure
- data/: dataset (CSV)
- code/: Python script and Jupyter Notebook
- results/: generated charts, tables, and analysis outputs

## How to Run
1. Place the dataset file in the data/ folder with the name:
   supermarket_sales.csv
2. Run the analysis script:
   python code/analysis.py

## Outputs
- Charts: results/charts/
- Tables: results/tables/
- Final answers: results/advanced_answers.txt
